# digi-checkbook

Simplify fixed expenses and see what flexibility remains.

## What it does

- Enter your income and fixed costs (housing, utilities, transportation)
- Set your own percentages for food, savings, crypto, and fun
- Instantly see how much you have for each category

## How to use

1. Open the HTML file in any browser
2. Fill in the numbers
3. Click Calculate

No installation. No data leaves your computer.

## Who it's for

Anyone who wants clarity on their monthly budget without complexity.

## Built with

Plain HTML, CSS, and JavaScript.
